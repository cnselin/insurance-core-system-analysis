# Insurance Core System – Business Analysis & System Design Simulation

## 📌 Overview

This repository contains a full-scale **Business Analysis & System Design simulation** for a mid-sized insurance company operating in Turkey.

The project focuses on designing a digital Insurance Core System MVP to replace Excel-based and manual processes.

> ⚠️ This is NOT a production-ready software implementation.  
> It is a professional enterprise-level BA & system design case study.

---

## 🎯 Problem Statement

Current challenges:

- Policy processes managed via Excel and email
- Manual underwriting approvals
- Slow claims handling
- No single source of truth
- Increasing regulatory pressure

---

## 🚀 Proposed Solution

Design of a modular Insurance Core System covering:

- Policy Management (creation, renewal, cancellation)
- Claims Management (FNOL, document handling, expert assignment, indemnity, payment)
- Customer Portal
- Agent/Broker Portal
- Underwriting workflow
- Rule-based Risk Scoring Engine
- Payment Integration
- Audit Logging
